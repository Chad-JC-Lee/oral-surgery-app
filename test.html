  • 右上角會顯示「🔑 API 設置」管理按鈕
  • 可以隨時查看、修改或清除 API Key

📝 手動測試

您也可以在瀏覽器控制台中執行以下代碼：

// 獲取 API Key console.log('API Key:', localStorage.getItem('google_ai_api_key')); // 設置 API Key localStorage.setItem('google_ai_api_key', 'your-test-api-key'); // 清除 API Key localStorage.removeItem('google_ai_api_key');
